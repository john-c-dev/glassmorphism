What is Glassmorphism?

Glassmorphism is a modern UI design trend that uses a frosted glass effect for elements. It features transparency, blur, and subtle borders to create a glass-like appearance.

Key Features

  • Transparent background
  • Background blur effect
  • Subtle light border
  • Minimal shadow effects

Glassmorphism CSS Code

.glass-element {
  /* Background with transparency */
  background: rgba(255, 255, 255, 0.15);
  
  /* Blur effect */
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);
  
  /* Border and shape */
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 15px;
  
  /* Shadow for depth */
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}
